Problem

你看到的票價，不是你付的票價

廉價航空的基礎票價只是開始，加上各種附加費後，實際花費可能遠超全服務航空。

🧳

托運行李費

廉航不含托運行李，20 kg 行李費依航線可達 NT$800–2,000，旺季更高。

💺

選位費

標準選位 NT$150–400，大空間座位最高 NT$1,500，前排座位另計。

🍱

機上餐點費

廉航餐點需另購，約 NT$300–500。全服務航空已含在票價中。

💳

信用卡手續費

部分廉航對信用卡付款加收 NT$200–300 手續費，容易被忽略。

Core Modules

三大核心功能

從真實資料查詢、費用計算到 AI 預測，一站式解決機票選購困境。

1

真實費用計算器

透過 fast_flights 即時抓取 Google Flights 的台灣籍航班資料（虎航、長榮、華航、星宇），結合航區費率表，自動計算每個需求組合的真實總費用，並比較是否買虎航 Bundle 方案更划算。

fast_flights Google Flights 費率資料庫 Bundle 偵測
2

ML 票價預測

以 Kaggle Clean_Dataset.csv（印度航班，300K+ 筆）訓練 RandomForestRegressor，學習 days_left → price_ratio 的漲跌規律，預測距出發 7 天後票價是否上漲，建議是否立即購票。

RandomForest price_ratio days_left scikit-learn
3

AI 推薦模組

整合 OpenAI GPT-4o-mini，根據最低費用航班的真實總成本、附加費明細與 ML 票價預測結果，生成繁體中文購票建議，幫助使用者做出最佳決策。

GPT-4o-mini OpenAI API 繁體中文 Rule-based Fallback

Machine Learning

票價預測模型設計

訓練資料來自印度航班（Kaggle），透過 price_ratio 正規化解決幣值差異，直接套用到台灣航班票價預測。

Price Ratio 正規化

訓練資料為印度盧比（INR），台灣票價為新台幣（TWD）。透過計算每條路線的中位數基準票價，將絕對票價轉換為相對倍率（price_ratio），消除幣值差異。

price_ratio = price / route_median

預測時，套用倍率變化到真實台灣票價：

predicted_TW = current_TW × (ratio_future / ratio_now)

模型：RandomForestRegressor
n_estimators=200, max_depth=12, R²≈0.89
